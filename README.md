# 🚗 Car Price Prediction using Machine Learning

This project predicts the resale price of cars using a Machine Learning model trained on historical car data.

The application allows users to input car specifications and receive an estimated price instantly through a web interface.

---

## 📌 Project Overview

The system uses a trained Machine Learning model to analyze car attributes and estimate a realistic resale price.

Features used include:

• Manufacturer  
• Production Year  
• Category  
• Fuel Type  
• Mileage  
• Gearbox Type  
• Airbags  
• Engine Volume  
• Levy  

---

## 🧠 Machine Learning Model

Algorithm Used:

Random Forest Regressor

Steps Performed:

• Data Cleaning  
• Feature Engineering  
• Label Encoding  
• Feature Selection (SelectKBest)  
• Model Training  
• Hyperparameter Tuning (GridSearchCV)
